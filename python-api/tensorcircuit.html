A class for compiled PCs. It is a subclass of torch.nn.Module.

Parameters:

  • root_ns (CircuitNodes) ‚Äì the root node of the PC‚Äôs DAG

  • layer_sparsity_tol (float) ‚Äì the maximum allowed fraction for added pseudo edges within every layer (better to set to a small number for sparse/block-sparse PCs)

  • max_num_partitions (Optional[int]) ‚Äì maximum number of partitions in a layer

  • disable_gpu_compilation (bool) ‚Äì force PyJuice to use CPU compilation

  • force_gpu_compilation (bool) ‚Äì force PyJuice to use GPU compilation

  • max_tied_ns_per_parflow_block (int) ‚Äì how many groups of tied parameters are allowed to share the same flow/gradient accumulator (higher values -> consumes less GPU memory; lower values -> potentially avoid stalls caused by atomic operations)

  • verbose (bool) ‚Äì Whether to display the progress of the compilation
