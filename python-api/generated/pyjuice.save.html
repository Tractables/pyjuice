pyjuice.save

pyjuice.save(fname: str, model: CircuitNodes | TensorCircuit)

Save a PC to file. The file should end with “.jpc”.

Parameters:

  • fname (str) – file name

  • model (Union[CircuitNodes,TensorCircuit]) – an uncompiled (CircuitNodes) or compiled (TensorCircuit) PC
